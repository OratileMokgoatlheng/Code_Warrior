What makes a code warrior?

You do not wake as a code warrior, instead you grow your in to becoming a code warrior. It takes practice, slepless nights, dedication and the eager to learn so that it is easy to be at that peak level.
In most cases to be good at something means you letting go of other things in life which can somehow limit you or hinder in successfully achieving your goal.
